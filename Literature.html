Литература для подготовки к олимпиадам

Р. К. Гордин «Геометрия. Планиметрия 7–9 классы».
Книга содержит задачи различной сложности по основным темам школьного курса планиметрии 7–9 классов.

В. В. Прасолов «Задачи по планиметрии».
Сборник содержит около 1900 задач с полными решениями и около 150 задач для самостоятельного решения.

А. В. Акопян «Геометрия в картинках».
Книга представляет собой сборник теорем классической геометрии, сформулированных в виде картинок.

О. Оре «Теория графов».
Книга содержит основные понятия и свойства графов.

О. И. Мельников «Теория графов в занимательных задачах».
Задачи из этой книги могут быть использованы при подготовке к математическим олимпиадам различных уровней.

Н. Х. Агаханов, И. И. Богданов, П. А. Кожевников, О. К. Подлипский, Д. А. Терешин «Всероссийская олимпиада школьников по математике 1993–2009».
Сборник задач Всероссийской олимпиады (муниципального и заключительного туров) по математике с ответами и полными решениями.

Л. Галицкий, А. М. Гольдман, Л. И. Звавич «Сборник задач по алгебре». Пособие для 7–9 классов,
в котором собраны задачи, способствующие систематическому углублению изучаемого материала и развитию навыков решения сложных задач.

З. И. Боревич «Определители и матрицы».
Пособие для тех, кто хочет познакомиться с началами линейной алгебры.
